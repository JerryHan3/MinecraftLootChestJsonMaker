# MinecraftLootChestJsonMaker

> [!Note]
> 中文文档见[README_cn.md](https://github.com/JerryHan3/MinecraftLootChestJsonMaker/blob/main/README_cn.md)

Generate a Minecraft loot table for Minecraft Wiki [Module:LootChest](https://zh.minecraft.wiki/w/Module:LootChest).

> [!Warning]
> The data generated by this script is not compatible with [English Minecraft Wiki](https://minecraft.wiki).

## Usage

1. Put both `JsonMaker.py` and `JsonMakerMap.json` in `loot_table` folder.
2. Configure map file to your need as mentioned in the section below.
3. Run the `JsonMaker.py`. The program will merge all JSON files from each folder specified in the map file, name it with the file name and specified suffix, and sort entries by name alphabetically.

A `result.json` and a `structureList.txt` file will be generated, containing the generated JSON table and the included structure list respectively.

### Map file formant

The map file `JsonMakerMap.json` is a compound list, each entry needs to contain following keys and values:

- `path` (required): a string of the relative path each folder of loot tables you want to include.
- `suffix` (required): a string of suffix to the name in final result.
- `overrides` (optional): a compound, set the key to the file name, and the value to the new suffix for that file.

The map file included in this repo is personally used for [Chinese Minecraft Wiki](https://zh.minecraft.wiki).
